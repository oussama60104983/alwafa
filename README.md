### Installation

1. Clone the repository:
```bash
git clone https://github.com/oussama60104983/alwafa.git
cd alwafa
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

The production build will be in the `dist` folder.
